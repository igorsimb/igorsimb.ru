Task and project management app

It supports one as well as groups of users working on the same projects. Users can also have different access levels.

Application is used to set up tasks and managing business projects. Main functions include:

  • Registration and authentication
  • Creating tasks and projects
  • User settings
  • Creating a company
  • Company tasks and user management
  • Dashboard
